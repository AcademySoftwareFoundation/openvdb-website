Detailed Description

This tool converts particles (with position, radius and velocity) into a signed distance field encoded as a narrow band level set. Optionally, arbitrary attributes on the particles can be transferred resulting in an additional attribute grid with the same topology as the level set grid.

Author
Ken Museth
Note
This fast particle to level set converter is always intended to be combined with some kind of surface post processing, i.e. tools::Filter. Without such post processing the generated surface is typically too noisy and blobby. However it serves as a great and fast starting point for subsequent level set surface processing and convolution.
