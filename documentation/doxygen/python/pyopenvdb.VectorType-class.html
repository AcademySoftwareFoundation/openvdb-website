Class VectorType

             VectorType

The type of a vector determines how transforms are applied to it.

  • INVARIANT: does not transform (e.g., tuple, uvw, color)
  • COVARIANT: apply inverse-transpose transformation with w = 0 and ignore translation (e.g., gradient/normal)
  • COVARIANT_NORMALIZE: apply inverse-transpose transformation with w = 0 and ignore translation, vectors are renormalized (e.g., unit normal)
  • CONTRAVARIANT_RELATIVE: apply "regular" transformation with w = 0 and ignore translation (e.g., displacement, velocity, acceleration)
  • CONTRAVARIANT_ABSOLUTE: apply "regular" transformation with w = 1 so that vector translates (e.g., position)
